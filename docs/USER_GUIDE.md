# User Guide

Welcome to Noctryx AI! This guide will help you navigate the platform as a student.

## 1. Registration & Login

- Navigate to `/register` to create an account.
- Fill in your name, email, and password.
- Once registered, you will be redirected to `/login` to sign in.

## 2. Using the AI Chat

- Click "New Chat" in the sidebar.
- Type your question about computer science, programming, or math in the input box.
- The AI will respond with a streaming, typing animation.
- You can upload code snippets or ask for explanations. Code blocks will have syntax highlighting and a copy button.

## 3. Managing Notes

- Go to Dashboard > Notes.
- Click "Create New Note".
- Write your note using Markdown formatting.
- Saved notes will appear in the grid below for easy access.

## 4. Flashcards

- Go to Dashboard > Flashcards.
- Create flashcards with a "Front" (question) and "Back" (answer).
- Click on any flashcard to flip it over and test your knowledge.

## 5. Assignments

- Go to Dashboard > Assignments.
- Add upcoming assignments and set their due dates.
- The dashboard will display them in chronological order to help you meet deadlines.
