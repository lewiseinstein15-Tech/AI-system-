# Deployment Guide

This guide covers deploying the Noctryx AI application using Docker and manual setups.

## Prerequisites

- A PostgreSQL database (e.g., Supabase, Neon, or self-hosted).
- An OpenAI API Key.
- Docker and Docker Compose installed (for Docker deployment).
